**O conversor universal de arquivos definitivo para o seu navegador.**

O **FileSwap** é uma ferramenta web elegante, rápida e segura, projetada para converter dados entre os formatos mais utilizados no mundo corporativo e de desenvolvimento: **CSV, JSON, XML e XLSX (Excel)**.

---

## ✨ Por que usar o FileSwap?

Diferente de outros conversores online, o FileSwap prioriza a **experiência do usuário** e a **segurança dos dados**:

*   **🔒 100% Privado:** Seus arquivos **nunca** saem do seu computador. Todo o processamento é feito localmente no seu navegador via JavaScript.
*   **📊 Prévia Interativa:** Visualize os dados antes de converter.
*   **✏️ Edição Direta:** Clique em qualquer célula da prévia para editar os dados antes de baixar a nova versão.
*   **📁 Múltiplas Abas (Excel):** Suporte total para arquivos `.xlsx` com seletor dinâmico de planilhas.
*   **🌓 Modo Escuro Nativo:** Interface moderna que se adapta à sua preferência de sistema.
*   **⚡ Ultra Rápido:** Conversões instantâneas, mesmo para arquivos grandes, graças ao uso de Web Workers e buffers de memória.

---

## 🚀 Como usar

1.  **Acesse o site:** Abra o link do GitHub Pages do projeto.
2.  **Carregue seu arquivo:** Arraste e solte seu arquivo na área indicada ou clique para selecionar.
3.  **Selecione a aba (opcional):** Se for um arquivo Excel com várias planilhas, escolha qual deseja converter.
4.  **Edite se necessário:** Dê um clique duplo em qualquer valor na tabela de prévia para alterá-lo.
5.  **Converta:** Escolha o formato de saída e clique em **"Converter e Baixar"**.

---

## 🛠️ Tecnologias Utilizadas

O FileSwap foi construído com o que há de mais moderno em desenvolvimento web *vanilla* (sem frameworks pesados):

*   **HTML5 & CSS3:** Com variáveis dinâmicas para temas e layout responsivo.
*   **Lucide Icons:** Para uma interface limpa e intuitiva.
*   **[SheetJS (XLSX)](https://sheetjs.com/):** O motor de processamento de planilhas mais robusto do mercado.
*   **[PapaParse](https://www.papaparse.com/):** O parser de CSV mais rápido para navegadores.
*   **[x2js](https://github.com/abdolence/x2js):** Conversão bidirecional entre XML e JSON de alta fidelidade.
